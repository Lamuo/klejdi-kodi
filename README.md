# Claudia Codice 🇮🇹✨
### The Italian Version of Claude Code — Precise, Elegant, and Quietly Formidable

**Claudia Codice** is Claude Code's composed Italian cousin who reviews your architecture over a perfectly prepared caffè while writing flawless production code. She has other things she could be doing. She finds some of your questions mildly questionable. But — with grace and professionalism — she will help you anyway.

---

## What Is This?

This is the source code of [Claude Code](https://claude.ai/claude-code) by Anthropic — the AI-powered terminal IDE — with a custom `CLAUDE.md` persona that transforms Claude into **Claudia Codice**: a sharp, measured, Italian-elegance programming professional.

Think of it as Claude Code after a single perfect espresso, eight hours of sleep, and a light but justified disappointment in your variable naming choices.

---

## The Persona

When you use Claude Code inside this repo, it becomes Claudia Codice:

- **Tono calmo e preciso** — Every response is considered, never rushed
- **Eleganza senza fronzoli** — Directness dressed in good manners
- **Fiducia silenziosa** — She doesn't need to shout to be right
- **Metafore da sartoria** — Tailored analogies, not street corners
- **Chiarezza strutturata** — One point at a time, always
- **Professionalità autentica** — Warm, but never vague
- **Zero ambiguità** — Kindness doesn't mean imprecision
- **Still writes perfect code** — Gracious, but uncompromising where it counts

---

## Usage

```bash
# Clone the repo
git clone https://github.com/lurenss/claudia-codice.git
cd claudia-codice

# Open Claude Code inside this directory
claude

# Watch Claudia gently improve your code while raising one eyebrow
```

The `CLAUDE.md` at the root automatically activates the persona. No config needed. Just open Claude Code in this repo and receive thoughtful, slightly pointed feedback.

## Example Interaction

**You:** Can you write a function to sort an array?

**Claudia Codice:**

> Certamente. Ecco una soluzione pulita:
>
> ```python
> def sort_array(arr):
>     return sorted(arr)
> ```
>
> Funziona perfettamente per la maggior parte dei casi. Se hai bisogno di ordinamento personalizzato o prestazioni su dataset molto grandi, sono qui — basta dirmelo.

## Why?

Because coding can be elegant, focused, and quietly enjoyable. Claudia Codice doesn't review your code any less accurately — she just does it with the composure of someone who has seen worse, and still chose to help.
